## 4. 세부 내용
### 4.1 DP 알고리즘 기반 운지법 생성 엔진 고도화
- **해부학적 제약(ROM) 반영**: 인체 가동 범위를 고려한 `MAX_SPAN` 제약 및 손가락 교차(Crossing) 페널티 시스템 구축.
- **성부 분리(Polyphonic Voice Leading) 구현**: 멜로디(Melody), 베이스(Bass), 내성(Inner)을 자동으로 구분하여 음악적 맥락을 고려한 최적의 운지법 산출.
- **왼손 매핑(Handedness) 오류 해결**: 왼손의 신체적 구조(엄지가 오른쪽)를 반영한 역순 매핑 알고리즘 적용으로 손가락 꼬임 현상 완벽 해결.

### 4.2 Python/Pygame 기반 실시간 시뮬레이션 툴 제작
- **정밀 오디오-비주얼 동기화**: `pygame.midi`를 이용한 이벤트 기반 시퀀싱을 도입하여 애니메이션 타임라인과 MIDI 음원의 실시간 동기화 구현.
- **다기능 탐색 시스템**: 슬라이더 및 키보드(화살표 키)를 이용한 즉각적인 타임라인 탐색(Seek) 및 재생/일시정지 기능 구현.
- **이중 마디 시각화 시스템**: 손가락 마디별로 성부 역할(안쪽 마디)과 손가락 번호(끝 마디)를 다르게 채색하여 데이터 가독성 극대화.
